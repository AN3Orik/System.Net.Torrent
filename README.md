System.Net.Torrent

Open source bittorrent scraper and peer wire implementation written in C#

- [x] UDP Announce
- [x] UDP Scrape
- [x] HTTP Announce
- [x] HTTP Scrape
- [-] PeerWire (TCP) Client (85%)
- [ ] Local Peer Discovery
